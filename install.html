git-crypt installation

Installation of git-crypt on Debian 10 buster:
  1. Install git-crypt
    $ apt-cache search git-crypt
    git-crypt - Transparent file encryption in git
    $ sudo aptitude install git-crypt
    [sudo] password for marcilr:
    The following NEW packages will be installed:

    0 packages upgraded, 1 newly installed, 0 to remove and 0 not upgraded.
    Need to get 74.9 kB of archives. After unpacking 201 kB will be used.
    Get: 1 http://deb.debian.org/debian buster/main amd64 git-crypt amd64 0.6.0-1 [74.9 kB]
    Fetched 74.9 kB in 0s (282 kB/s)
    Selecting previously unselected package git-crypt.
    (Reading database ... 384492 files and directories currently installed.)
    Preparing to unpack .../git-crypt_0.6.0-1_amd64.deb ...
    Unpacking git-crypt (0.6.0-1) ...
    Setting up git-crypt (0.6.0-1) ...
    Processing triggers for man-db (2.8.5-2) ...

    $

  2. Create a new git repository, with git init, or change into an existing repo with cd myrepo
  3. Update repository to use git-crypt with:

    $ cd myrepo
    $ git-crypt init

    Observe output:

    Generating key...

    Save a copy of the git-crypt symmetric key the was generated for the repository to the next higher directory:

    $ git-crypt export-key ../git-crypt-key
